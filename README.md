# NLP_Mini_Project
News Article Topic Classification using Deep Learning (DistilBERT). The model classifies news headlines and descriptions into 40+ categories with high accuracy and includes a Streamlit-based UI for real-time prediction.


We have individually used two different approches one is (DistilBert) and second one  (TF-IDF + SVM ) .


DistilBert  = 71 % accuracy
(TF-IDF + SVM ) = 81.53 % accuracy
